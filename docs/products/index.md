# Products — Technical Documentation

This section provides comprehensive product documentation for every paprika product offered by Dinweys (Qingdao). Co., Ltd. Each product page includes specifications, analytical data, storage requirements, and applications.

## Product Overview

Our product line covers four grades across two primary forms (powder and flakes), plus whole pods and oleoresin feedstock material. This comprehensive range serves the full spectrum of B2B applications — from gourmet seasoning blends to industrial oleoresin extraction.

### Grade Classification System

Paprika grades are primarily classified by ASTA color value, the international standard for color quality measurement:

| Grade Classification | ASTA Color Value | Relative Price Index | Market Position |
|:-------------------:|:----------------:|:--------------------:|:----------------|
| Special | 200+ | 1.5–2.0× | Niche premium |
| Premium | 160–200 | 1.2–1.5× | Top tier |
| Superior | 120–160 | 1.0× (Baseline) | Mid-premium |
| Standard | 80–120 | 0.7–0.85× | Cost-effective |

### Application Guide

| Application | Recommended Grade | Form | Rationale |
