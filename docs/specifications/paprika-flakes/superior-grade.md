# Paprika Flakes — Superior Grade Specification

## Product Identification

| **Product Information** | |
|-------------------------|---|
| Product Name | Paprika Flakes Superior Grade |
| Product Code | PF-SUP-001 |
| Grade | Superior |
| Product Form | Flakes (coarse cut) |
| Botanical Source | *Capsicum annuum* L. |
| Intended Market | EU, US, general food manufacturing, bulk seasoning |
| Regulatory Status | Food-grade, GRAS 21 CFR 182.10 |
| HS Code | 0904.21.00 (dried, neither crushed nor ground) |
| Country of Origin | China (Xinjiang / Shandong) |

**Description:** Superior Grade paprika flakes offer a balanced combination of visible piece identity and good red color (ASTA 120–160) at a cost-effective price point for volume buyers. Produced from *Capsicum annuum* L. pods, coarse-cut to 4–8 mesh with controlled fines. Preferred for general seasoning blends, processed meat rubs, and foodservice bulk where paprika flake identity is desired without the premium price tag.

## Application Notes

Superior Grade flakes (ASTA 120–160) serve as the **standard flake choice** for volume seasoning and processed food manufacturing.

| Application Segment | Suitability | Notes |
|---------------------|-------------|-------|
| Bulk seasoning blends | ★★★★★ | Best price-to-visual-impact ratio for flakes |
| Processed meat rubs | ★★★★★ | Adheres well; visible at serving |
| Foodservice seasoning | ★★★★☆ | Good quality for restaurants and canteens |
| Snack seasonings | ★★★☆☆ | Powder preferred for adhesion; flakes may separate |
| Retail value lines | ★★★★☆ | Acceptable for mid-tier retail products |
| Dry soup mixes | ★★★☆☆ | Limited; fine powder preferred for instant soups |

**Best For:** Volume seasonings, processed meat and poultry rubs, foodservice bulk seasoning, mid-tier retail blends.
