# Paprika Flakes — Standard Grade Specification

## Product Identification

| **Product Information** | |
|-------------------------|---|
| Product Name | Paprika Flakes Standard Grade |
| Product Code | PF-STD-001 |
| Grade | Standard |
| Product Form | Flakes (coarse cut) |
| Botanical Source | *Capsicum annuum* L. |
| Intended Market | China domestic market, Middle East, Africa, price-sensitive markets |
| Regulatory Status | Food-grade, compliant with China GB standards |
| HS Code | 0904.21.00 (dried, neither crushed nor ground) |
| Country of Origin | China (Xinjiang / Gansu) |

**Description:** Standard Grade paprika flakes deliver functional flake identity at the most economical price point. With ASTA color value of 80–120 and a wider SHU tolerance (≤2,000), this grade is designed for cost-sensitive manufacturing environments and markets where moderate color intensity is acceptable. Coarse-cut to 4–8 mesh with controlled fines, providing visual piece identity in bulk seasoning applications.
