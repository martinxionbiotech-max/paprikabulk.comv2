# Paprika Grade Comparison Chart

A comprehensive side-by-side comparison of the four Dinweys paprika powder and four flake grades, plus whole pods. Use this reference to select the optimal grade for any application.

---

## Quick Selection Guide

| Your Priority | Recommended Grade(s) |
|---------------|---------------------|
| Maximum color intensity (ASTA 200+) | Special Grade (powder or flakes) |
| Oleoresin / color extraction yield | Special Grade Powder |
| Premium consumer-facing product | Premium Grade (powder or flakes) |
| Best price-to-performance ratio | Superior Grade (powder or flakes) |
| Lowest cost per kg | Standard Grade (powder or flakes) |
| Visible piece identity | Flakes (any grade; Premium best color) |
| Whole-pod import / on-site processing | Whole Pods |
| Blending to boost color value | Special Grade as minor fraction |

## Grade Selection Guide

| Your Need | Recommended Grade | Rationale |
|-----------|-------------------|-----------|
| Maximum color intensity | **Special ASTA200+** | Highest capsanthin concentration |
| Premium food manufacturing | **Premium** | Best balance of color, low SHU, and quality perception |
| Balanced cost and quality | **Superior** | Optimal price-to-performance ratio |
| Large volume, standard color | **Standard** | Lowest cost per ASTA unit |
| Oleoresin extraction | **Special ASTA200+ Powder** | Highest oleoresin yield per kg |
| Retail packaging (consumer) | **Premium or Special** | Color drives purchase decisions |
| Snack seasonings (premium) | **Premium Powder** | Fine grind, bright color |
| Snack seasonings (value) | **Superior or Standard Powder** | Economical with higher loading |
| Visible piece identity | **Flakes (any grade)** | Coarse cut provides visual appeal |
| Gourmet/decorative garnish | **Special Flakes** | Deep red color commands premium |
| Whole-pod import/on-site grinding | **Whole Pods** | Tariff and freight advantages |
| Blending to target ASTA | **Special Grade as booster** | Small % lifts blend ASTA significantly |
| Meat processing (sausages) | **Superior Powder** | Standard industry choice |
| Rubs & marinades (dry) | **Superior or Premium Flakes** | Adherence + piece identity |
| Cost-sensitive bulk | **Standard (powder or flakes)** | Best unit economics |
| EU regulatory compliance | **Premium, Superior, or Special** | EU MRL default |
| China domestic market | **Standard** | China GB standard default |
